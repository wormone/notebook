气象数据处理和绘图常用的python软件包

January 2017 · 1 minute read

1. 基本问题

- datetime, canlendar：时间和日期处理
- re：正则处理
- glob：文件名匹配
- os：系统路径等操作

2. 特定格式

- netCDF4：netCDF数据读写
- pygrib：grib数据读写
- struct：二进制数据读写
- gdal：geotiff等数据读写

3. 计算分析

- numpy, scipy：科学计算
- pandas：数据结构框架
- scikit-learn：统计学和机器学习模型

4. 绘图

- matplotlib, basemap：几乎所有类别的气象图形
- seaborn：更美观一些的图形

5. 业务自动化

- apscheduler：任务管理
- logging：日志记录
- pyWRF：WRF管理
